-------  DSK File Converter  --------

◆本ソフトウェアについて

    画像ファイル、音楽ファイル、動画ファイルの形式を変更することができます。


-------動作環境-------

  推奨動作環境

    OS                Microsoft Windows 10 64bit

    ハードウェア構成  OSのシステム要件に準ずる


-------ファイルアクセス-------

  ・実行ファイル設置フォルダ
  ・保存フォルダ(設定で変更)
  ・読み込むファイルの設置フォルダ

-------使用方法-------

    ①画像、音楽、動画いずれかのフォーマット変換の横にある「ファイルを開く」ボタンを押す。
    ②ファイルを選択し(複数選択可)、拡張子を指定する。
    ③「変換」ボタンを押し、出力まで待つ。
